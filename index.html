Image Homography Estimation

Implemented homography estimation to rectify images containing planar surfaces captured at an angle. This enables perspective correction and document rectification.

Key Achievements

  • Custom DLT (Direct Linear Transform) implementation with point normalization.
  • Inverse mapping with bilinear interpolation for hole-free warping.
  • Interactive point selection tool for correspondence marking.
  • Comparison with OpenCV implementation (perfect match achieved).

Warping Methods Comparison

Comprehensive comparison between Triangular Mesh Warping and Thin-Plate Spline (TPS) Warping, analyzing their differences in smoothness, continuity, and computational cost.

Comparative Analysis

Aspect Triangular Mesh Warping TPS Warping
Smoothness Piecewise smooth (C^0 continuous) Globally smooth (C^infinity continuous)
Speed Fast (O(T*P)) Slower (O(N^3 + M*N))
Discontinuities May occur at triangle boundaries No discontinuities
Local Control Excellent Global influence
Best Use Cases Real-time applications, AR/VR Medical imaging, animation

Key Findings

  • Performance: Triangular mesh is 5-20x faster than TPS.
  • Quality: TPS produces smoother, more natural-looking deformations.
  • Application: Choose based on speed versus quality requirements.
  • Complexity: TPS relies on O(N^3) matrices while mesh warping uses local O(T*P) transforms.
